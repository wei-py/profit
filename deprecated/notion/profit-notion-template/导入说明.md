# Profit 商品利润管理模板 — 导入说明

## 导入方式一：Notion ZIP 导入（推荐）

1. 打开 [Notion](https://www.notion.so)
2. 左侧点击 `Import`
3. 选择 `Text & Markdown / CSV`
4. 上传整个 `profit-notion-template.zip` 文件
5. Notion 会自动识别 Markdown 页面和 CSV 表格
6. 导入后把所有数据库移动到 `Profit 商品利润管理模板` 页面下

## 导入方式二：逐个 CSV 导入

1. 打开 [Notion](https://www.notion.so)
2. 左侧点击 `Import` → `CSV`
3. 依次导入以下文件：
   - 字段库.csv
   - 规则库.csv
   - 需求池.csv
   - 版本日志.csv
4. 创建一个新页面，标题为 `Profit 商品利润管理模板`
5. 把 `Profit 商品利润管理模板.md` 复制到该页面
6. 把 4 个导入的数据库也移到该页面下

## 导入后调整字段类型

Notion CSV 导入后字段类型默认为 Text，建议调整为更合适的类型：

### 字段库
- `层级` → Select：平台级 / 商品级 / SKU级
- `输入输出` → Select：输入 / 输出
- `类型` → Select：文本 / 数字 / 下拉 / 布尔 / 日期

### 规则库
- `规则排序` → Number
- `启用` → Checkbox
- `规则类型` → Select：查表 / 映射 / 计算 / 条件 / 输出

### 需求池
- `状态` → Select：待处理 / 进行中 / 已完成 / 暂缓
- `优先级` → Select：高 / 中 / 低
- `模块` → Select：商品页 / 配置页 / 规则编辑器 / 更新发布 / Excel配置
- `完成时间` → Date

### 版本日志
- `发布时间` → Date
- `状态` → Select：计划中 / 已发布 / 回滚
- `验证状态` → Select：未验证 / 通过 / 有问题

## 如果需要建立 Relation（可选）

手动在 Notion 中建立关联字段可以为以下属性建立 Relation：

- 规则库 `输入字段` → 字段库
- 规则库 `输出字段` → 字段库
- 需求池 `关联版本` → 版本日志
- 版本日志 `关联需求` → 需求池
- 规则库 `关联需求` → 需求池

## 如果您想做成可售卖模板

1. 导入并调整完所有字段后
2. 在 Notion 右上角点 `Share`
3. 点 `Publish`
4. 勾选 `Allow duplicate as template`
5. 把生成的链接作为商品链接发给买家
