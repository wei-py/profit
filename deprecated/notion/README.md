# Notion 资料

这里集中放 Profit 项目相关的 Notion 资料。

## 文件说明

- `profit-notion-template.zip`：可直接导入 Notion 的 ZIP 包，包含首页、4 个 CSV 数据库和导入说明。
- `profit-notion-template/`：ZIP 包的源文件目录。
- `copy-paste.md`：可复制到 Notion 的参考页面内容（备用）。
- `version-1.5.1.md`：版本 1.5.1 的更新日志。
- `import/`：CSV 文件单独存放目录（已打包进 ZIP 包）。

## 如何使用 profit-notion-template.zip

直接在 Notion 中：

```text
Import → Text & Markdown / CSV → 选择 profit-notion-template.zip
```

导入后把数据库移到同一页面，按 `导入说明.md` 调整字段类型（Select / Date / Checkbox / Relation）。

## 如果要做成可售卖 Notion 模板

1. 导入 `profit-notion-template.zip` 到 Notion
2. 在 Notion 里检查并调整所有字段类型
3. 根据需要调整视图、页面布局、模板按钮
4. 点右上角 `Share → Publish → Allow duplicate as template`
5. 把 Notion 链接作为正式商品交付

ZIP 包只是帮你快速初始化数据库结构，最终成品是 Notion 页面链接。
