# 项目

## 要求

- 使用一份 `配置.xlsx` 统一管理预设页、选项页、模板页。
<!-- - 不沿用当前项目实现，后续由其他 AI 按本文档重新编写。 -->
- `配置.xlsx` 只负责配置数据，不负责商品记录数据。
- `配置.xlsx` 是预设页、选项页、模板页的唯一配置数据源。
- 创建页读取的是 `配置.xlsx` 中的预设、选项、模板。
- 列表页使用独立的列表 Excel，不写回 `配置.xlsx`。

- 预设页：用于按国家和平台维护组合，并管理该组合下的参数项。
- 选项页：用于维护可复用的选项分组和选项值。
- 模板页：用于维护规则表及其匹配条件和结果值。
  - 巴西规则参考：https://www.mercadolivre.com.br/ajuda/quanto-custa-vender-um-produto_1338
- 创建页：用于录入商品信息，并结合预设和模板生成计算结果。
- 列表页：用于管理已加入列表的商品记录，并同步到列表 Excel。

## 预设页补充

- 主要操作：新建组合、搜索组合、组合列表。
- 预设页管理组合下的参数项。
- 选项页管理参数项里可复用的枚举值。
- 当预设参数项需要固定候选值时，应优先由选项页提供。
- 参数项使用可编辑表格维护。
- 表格字段包括：参数名称、类型、单位、值 / 规则、操作。
- 支持新增、修改、删除参数项，以及补齐默认项。
参考 excel-rule-engine-example.xlsx

## 选项页补充

- 主要操作：新增分组、切换分组、新增选项。
- 选项页用于维护可复用的枚举值。
- 一个分组下可维护多条选项。
- 选项使用可编辑表格维护。
- 表格字段包括：显示名称、实际值、排序、启用、备注、操作。
- 支持新增、修改、删除选项。
- 预设页、模板页、创建页中需要固定候选值的字段，应优先复用选项页分组。
参考 excel-rule-engine-example.xlsx

## 创建页补充

- 创建页用于录入当前商品信息，并生成计算结果。
- 创建页会读取当前预设组合的参数项。
- 创建页会读取选项页中的可复用枚举值。
- 创建页会读取模板页中的规则模型。
- 创建页维护的是当前商品的实际输入值，不是默认配置。
- 创建页中的字段分为基础信息、计算字段、结果字段、图片、变体。
- 基础信息包括名称、款号、成本、重量等。
- 创建页中的图片支持上传、本地路径或链接，并可与变体和选项关联。
- 可选字段优先复用选项页分组。
- 规则结果字段通过模板查表或规则计算得到。
- 创建页应支持把当前结果保存为一条商品记录，并加入列表页。

## 列表页补充

- 列表页用于管理已加入列表的商品记录。
- 列表页展示的是商品记录结果，不负责重新配置规则。
- 列表页应支持绑定列表 Excel。
- 列表页应支持读取、展示、删除、保存商品记录。
- 列表页中的记录来自创建页添加的结果。
- 一条记录应包含基础信息、结果字段、图片、变体和扩展字段。
- 列表页中的图片应支持预览。
- 列表页保存时，应同步到列表 Excel，而不是主规则 Excel。
