# 表格内部滚动 + 列表内部滚动 + 高度自动到底 (修复版)

## Root Cause

两个 CSS 错误导致高度链断裂：

### Bug 1: Tailwind v4 `flex` 不是 flex-grow

在 Tailwind v4 中：
- `flex` = `display: flex`（不是 `flex: 1 1 0%`）
- `flex-1` = `flex: 1 1 0%`

当前代码使用 `card-body flex flex-col min-h-0`，只设置了 `display: flex` + `flex-direction: column`，但 **未覆盖 daisyUI 的 `card-body { flex: 1 1 auto }`**。`flex-basis: auto` 使 card-body 按内容尺寸计算，不会填充 card 的剩余高度。

**修复**: `card-body flex flex-col min-h-0` → `card-body flex-1 flex-col min-h-0`

### Bug 2: EditableTable 根 div 未填充父容器高度

`<div class="flex-1 min-h-0">` 包裹 EditableTable 只是一个普通块元素，不是 flex 容器。EditableTable 的根 `<div class="flex flex-col min-h-0">` 作为普通块子元素，不会自动拉伸到父容器高度（div 默认 `height: auto`）。

**修复**: EditableTable 根 div 加 `h-full` → `<div class="h-full flex flex-col min-h-0">`  
（父容器 `flex-1` 有确定高度，`h-full` 能正确解析）
