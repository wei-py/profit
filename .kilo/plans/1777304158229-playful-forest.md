# Driver.js 用户引导集成

## 概述

集成 [Driver.js](https://driverjs.com) 为"利润工具"添加交互式用户引导教程，通过侧边栏按钮手动触发，覆盖全部 6 个页面。

## 实现步骤

### 1. 安装 Driver.js

```bash
npm install driver.js
```

### 2. 创建 `src/composables/useTour.js`

- 导入 `driver` 和 `driver.js/dist/driver.css`
- 定义 `tourRoutes` 按页面划分的步骤配置（中英文混合文案）
- 导出 `startTour(routeName)` 和 `stopTour()`
- `startTour` 根据当前路由名启动对应页面引导，若传 `'all'` 则启动全局引导（跨页面导航）
- 步骤使用 `data-tour="xxx"` 属性选择器定位元素

### 3. 修改 `DefaultLayout.vue`

- 在侧边栏底部（主题切换按钮下方）添加"帮助 ?"按钮
- 按钮点击时调用 `startTour(currentRoute)`
- 导入 `useTour`

### 5. 引导步骤文案设计

#### 侧边栏（公共）
| 步骤 | 标题 | 描述 |
|------|------|------|
| 导航菜单 | 页面导航 | "利润工具"共有6个功能页面：预设、选项、字段、模板、新建、列表。点击即可切换。 |
| 主题切换 | 主题切换 | 点击可切换浅色/深色模式，适合不同光线环境使用。 |
| 侧边栏折叠 | 收起侧边栏 | 点击汉堡菜单可折叠侧边栏，为主内容区腾出更多空间。 |

#### 预设页
| data-tour | 标题 | 描述 |
|-----------|------|------|
| preset-toolbar | 打开配置 / 新建预设 | 首次使用需先"打开配置"Excel文件。已有配置后可"新建预设"并"保存配置"。 |
| preset-search | 搜索预设 | 输入关键词可快速筛选预设列表，支持按名称、国家、平台搜索。 |
| preset-list | 预设列表 | 展示所有已创建的预设，点击可查看和编辑其参数。 |
| preset-param-table | 预设参数 | 为预设定义参数列表，可添加、编辑、删除参数。点击"补全默认参数"可快速导入字段。 |

#### 选项页
| data-tour | 标题 | 描述 |
|-----------|------|------|
| option-toolbar | 管理选项分组 | 选项分组用于为"选择型"字段提供下拉选项，如颜色、尺寸等。 |
| option-group-list | 分组列表 | 点击分组可查看和编辑其下的选项项。 |
| option-item-table | 选项项 | 为分组添加具体选项，设置显示名、值和排序。 |

#### 字段页
| data-tour | 标题 | 描述 |
|-----------|------|------|
| field-table | 字段定义 | 定义所有可用字段的键、名称、类型等信息。字段会在预设参数和规则条件中被引用。 |

#### 模板页
| data-tour | 标题 | 描述 |
|-----------|------|------|
| template-ruleset-list | 规则集 | 选择规则集，一个规则集包含多条规则。 |
| template-rule-list | 规则列表 | 规则定义了什么条件下执行什么动作，按优先级排序。 |
| template-tabs | 条件与动作 / 查找表 | 切换编辑规则的条件动作，或管理查找表数据。 |
| template-condition-tree | 条件树 | 以树形结构组织条件，支持 AND/OR 逻辑组合，可嵌套分组。 |
| template-action-list | 动作列表 | 条件满足时执行的动作，支持赋值、查表、分支三种类型。 |
| template-lookup-table | 查找表 | 用于"查表"动作匹配的数据表，支持精确匹配和区间匹配。 |

#### 新建页
| data-tour | 标题 | 描述 |
|-----------|------|------|
| create-preset-select | 选择预设 | 选择一个已启用的预设作为计算模板。 |
| create-basic-info | 基础信息 | 填写产品名称、款号、成本和重量等基本信息。 |
| create-params | 输入参数 | 根据预设定义的参数填写具体数值。 |
| create-calculate | 执行计算 | 点击"计算"按钮，系统将根据预设规则自动计算利润等结果。 |
| create-results | 计算结果 | 查看计算结果，可点击"保存到列表"将记录存入列表页。 |

#### 列表页
| data-tour | 标题 | 描述 |
|-----------|------|------|
| list-toolbar | 操作列表 | 打开或加载列表 Excel 文件，保存修改。 |
| list-table | 记录列表 | 展示所有保存的记录，支持查看详情、编辑和删除操作。 |

## 风险与注意

- Driver.js 的 CSS (`driver.css`) 与 DaisyUI 样式不会冲突，Driver.js 使用独立的 `.driver-` 命名空间
- `data-tour` 属性仅在 Vue 模板中使用，不参与任何运行时逻辑，无性能影响
- 引导步骤仅在用户点击"帮助"按钮时触发，不会自动弹出，避免干扰

## 验证方式

1. `npm run build` 确保构建通过
2. 启动应用后点击侧边栏"帮助"按钮，验证各页面引导正常显示
3. 检查 Driver.js 弹层样式是否与主题协调
