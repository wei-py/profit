## 1. 预设选中持久缓存

**文件**: `src/pages/PresetPage.vue`

- `selectedPresetId` 当前是 `ref('')`，页面刷新后丢失
- 使用 `localStorage` 持久化（key: `profit-selected-preset-id`）
- **onMounted**: 从 localStorage 恢复 `selectedPresetId`；需校验该 preset 在 `configStore.config.presets` 中仍然存在
- **watch `selectedPresetId`**: 值变化时写入 localStorage（空字符串也写，允许清除缓存）

```js
import { onMounted, watch } from 'vue'

const CACHE_PRESET_KEY = 'profit-selected-preset-id'

onMounted(() => {
  if (!configStore.loaded) return
  const cached = localStorage.getItem(CACHE_PRESET_KEY)
  if (cached && configStore.config.presets.some(p => p.presetId === cached)) {
    selectedPresetId.value = cached
  }
})

watch(selectedPresetId, (val) => {
  localStorage.setItem(CACHE_PRESET_KEY, val)
})
```

**与 `useFileIO.js` 中现有缓存模式一致**: 项目已有 `profit-theme`（主题）和 `lastConfigPath`（上次配置文件路径）使用 localStorage。新增 key 遵循 `profit-*` 命名约定。
