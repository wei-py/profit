# 深色/浅色主题切换

## 现状

- daisyUI v5 已定义两个自定义主题：`me`（浅色）和 `me-dark`（深色）
- 当前 `style.css` 使用 `--prefersdark` 标志，由操作系统偏好自动决定深/浅色
- 无手动切换 UI，用户无法主动控制主题

## 修复方案

### 1. `src/style.css` — 移除 `--prefersdark` 标志

将 `me-dark --prefersdark` 改为 `me-dark`，关闭 daisyUI 的自动检测，改由 JS 手动控制 `data-theme` 属性。

### 2. `src/composables/useTheme.js` — 新建主题 composable

- 使用 Tauri plugin-store（`@tauri-apps/plugin-store`）持久化主题偏好
- 浏览器开发模式回退到 `localStorage`
- 初始化时读取持久化值并设置 `<html data-theme="...">`
- 提供 `theme` (ref)、`isDark` (computed)、`toggleTheme()` 方法

### 3. `src/App.vue` — 初始化主题

在 `onMounted` 中调用 `useTheme()` 触发主题初始化。

### 4. `src/layouts/DefaultLayout.vue` — 添加切换按钮

在侧边栏底部添加主题切换按钮：
- 浅色时显示月亮图标 → 点击切换到深色
- 深色时显示太阳图标 → 点击切换到浅色
- 侧边栏折叠时只显示图标，展开时显示图标 + 文字
