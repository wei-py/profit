# 全局字体调大

## 现状

- 项目使用 Tailwind CSS v4 + daisyUI v5，所有字体大小通过 Tailwind 的 `text-*` 工具类控制
- 当前无任何 base `font-size` 声明，完全依赖浏览器默认的 `16px` 作为 `1rem` 基准
- 页面中大量使用 `text-xs`（12px）、`text-sm`（14px），中文阅读普遍偏小

## 方案

在 `src/style.css` 的 `html` 选择器中设置 `font-size: 17px`（默认 16px → 17px），全局放大 ~6.25%，所有 `rem` 单位的字体等比缩放。改动仅 1 行，影响整个应用。

## 涉及文件

| 文件 | 变更 |
|------|------|
| `src/style.css` | `html` 块中加 `font-size: 17px` |
