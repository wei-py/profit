# 预设页类型列中文化 + 下拉选择时默认值联动

## Overview

2 项改进:
1. 类型列显示中文（文本/数字/下拉选择/布尔/日期）
2. 当类型为"下拉选择"时，默认值列自动切换为下拉框，选项来自关联的选项组

## Files to Change

| File | Changes |
|------|---------|
| `src/components/common/EditableTable.vue` | 扩展列能力：view mode 显示 select label、edit mode 支持动态 type/options |
| `src/pages/PresetPage.vue` | 修改 `paramColumns`：type 列加中文 options，defaultValue 列加动态 type/options |

## Discovery

- **EditableTable 现有结构**：`col.type` 支持 `select`/`boolean`/`number`/`textarea`，`col.options` 为静态数组。view mode 中 `select` 列无特殊处理，直接显示原始值。
- **FieldInput 参考实现** (`src/components/common/FieldInput.vue:12-22`)：已有通过 `optionGroupId` 查找选项组并过滤 `enabled !== false` 项的逻辑，可直接复用模式。
- **支持的字段类型**: `text`、`number`、`select`、`boolean`、`date`（从 `FieldInput.vue` 和 `validate.js` 确认）。

## Implementation

### 1. EditableTable.vue — 扩展列定义

#### 1a. 新增辅助函数

```js
function getEffectiveType(row, col) {
  if (col.getType) return col.getType(row)
  return col.type || 'text'
}

function getEffectiveOptions(row, col) {
  if (col.getOptions) return col.getOptions(row) || []
  return col.options || []
}
```

#### 1b. View mode — select 列显示中文标签

当前行 93-101：select 列在 view mode 直接显示原始值 `{{ getValue(row, col) }}`。

在 `<template v-else>` 前插入：
```html
<template v-else-if="col.type === 'select' && col.options">
  {{ col.options.find(o => o.value === getValue(row, col))?.label || getValue(row, col) }}
</template>
```

#### 1c. Edit mode — 支持动态 type 和 options

当前行 60-66：`col.type === 'select' && col.options` 硬编码为 select 渲染。

替换为 effective 检测，覆盖 select 列的第一条判断：
```html
<td v-for="col in columns" :key="col.key">
  <template v-if="getEffectiveType(editData, col) === 'select'">
    <select v-model="editData[col.prop || col.key]" class="select select-bordered select-sm w-full">
      <option value=""></option>
      <option v-for="opt in getEffectiveOptions(editData, col)" :key="opt.value" :value="opt.value">
        {{ opt.label }}
      </option>
    </select>
  </template>
  <template v-else-if="getEffectiveType(editData, col) === 'boolean'">
    <input v-model="editData[col.prop || col.key]" type="checkbox" class="toggle toggle-sm">
  </template>
  <template v-else-if="getEffectiveType(editData, col) === 'number'">
    <input v-model.number="editData[col.prop || col.key]" type="number" class="input input-bordered input-sm w-full">
  </template>
  <template v-else-if="getEffectiveType(editData, col) === 'textarea'">
    <textarea v-model="editData[col.prop || col.key]" class="textarea textarea-bordered textarea-sm w-full" rows="2" />
  </template>
  <template v-else>
    <input v-model="editData[col.prop || col.key]" type="text" class="input input-bordered input-sm w-full">
  </template>
</td>
```

关键变化：
- `select` → `getEffectiveType(editData, col) === 'select'`（覆盖静态 + 动态 select）
- `boolean` → `getEffectiveType(editData, col) === 'boolean'`
- `number` → `getEffectiveType(editData, col) === 'number'`
- select options 从 `col.options` 改为 `getEffectiveOptions(editData, col)`

### 2. PresetPage.vue — paramColumns

修改 `paramColumns` 数组（当前行 92-100）：

```js
const paramColumns = [
  { key: 'paramName', prop: 'paramName', label: '名称' },
  {
    key: 'type', prop: 'type', label: '类型',
    type: 'select',
    options: [
      { value: 'text', label: '文本' },
      { value: 'number', label: '数字' },
      { value: 'select', label: '下拉选择' },
      { value: 'boolean', label: '布尔' },
      { value: 'date', label: '日期' },
    ],
  },
  { key: 'unit', prop: 'unit', label: '单位' },
  { key: 'optionGroupId', prop: 'optionGroupId', label: '选项组' },
  {
    key: 'defaultValue', prop: 'defaultValue', label: '默认值',
    getType: (row) => row.type === 'select' ? 'select' : 'text',
    getOptions: (row) => {
      if (row.type !== 'select' || !row.optionGroupId) return []
      const group = configStore.getOptionGroup(row.optionGroupId)
      if (!group?.items) return []
      return group.items.filter(i => i.enabled !== false).map(i => ({
        value: i.itemValue,
        label: i.itemLabel || i.itemValue,
      }))
    },
  },
  { key: 'sort', prop: 'sort', label: '排序', type: 'number' },
  { key: 'isRequired', prop: 'isRequired', label: '必填', type: 'boolean' },
]
```
